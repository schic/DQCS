Interface HasKeyPreview

Deprecated. This interface is no longer implemented by any GWT library. Use Event.addNativePreviewHandler(com.google.gwt.user.client.Event.NativePreviewHandler) instead. 

@Deprecated
public interface HasKeyPreview

A class that implements this interface receives a preview of keyboard events before they are passed to the focused widget.
