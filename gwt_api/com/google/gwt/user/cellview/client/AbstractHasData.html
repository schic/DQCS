Set a values associated with the rows in the visible range.

This method does not replace all rows in the display; it replaces the row values starting at the specified start index through the length of the the specified values. You must call HasRows.setRowCount(int) to set the total number of rows in the display. You should also use HasRows.setRowCount(int) to remove rows when the total number of rows decreases.

Set the SelectionModel used by this HasData.

If you need finer control over selection, you can specify a DefaultSelectionEventManager using setSelectionModel(SelectionModel, com.google.gwt.view.client.CellPreviewEvent.Handler).
