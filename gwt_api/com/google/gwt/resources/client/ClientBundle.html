public interface ClientBundle

The use of this interface is similar to that of ImageBundle. Declare no-argument functions that return subclasses of ResourcePrototype, which are annotated with ClientBundle.Source annotations specifying the classpath location of the resource to include in the output. At runtime, the functions will return an object that can be used to access the data in the original resource.
