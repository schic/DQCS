Interface NumberConstants

NumberConstantsImpl
public interface NumberConstants

NumberConstants class encapsulate a collection of Number formatting symbols for use with Number format and parse services. This class extends GWT's Constants class. The actual symbol collections are defined in a set of property files named like "NumberConstants_xx.properties". GWT will will perform late binding to the property file that specific to user's locale. If you previously were using GWT.create on this interface, you should use LocaleInfo.getCurrentLocale().getNumberConstants() instead.
